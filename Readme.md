# Currency Converter

```
Simple GRPC to convert Currency. 
```

## How To Run:
```
>   Open Terminal
>   git clone git@github.com:Manan-Rastogi/CurrencyConverter.git
>   cd CurrencyConverter
>   Open 2 Terminals
>   In First Terminal:
    >>  cd server
    >>  go run .
>   In Second Terminal:
    >>  cd client
    >>  go run .
>   In client folder, in main.go check and comment and uncomment which stream you want to run.
```
